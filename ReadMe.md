# Simple Angular Example For beginner With Json Server

# Useful Commands

## Json server Related

json-server --watch users.json
## Angular

### Install

	npm install -g @angular/cli
	ng new my-dream-app
	cd my-dream-app

### Compile

	ng s -O


### Module Add 

	ng generate component components/layout/heade
	ng g c "componenet name"

## Service Add

	ng g s "service name"
